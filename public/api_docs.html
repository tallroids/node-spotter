Spotter REST API Documentation

Locations

/api/getLocations

Returns all public locations

/api/getLocationById/:id

Returns a location object for specified id

/api/getLocationsByCategoryId/:id

Returns all public locations that are categorized by specified id

/api/createLocation

Creates a location with the following parameters:

  • title (TEXT)
  • description (TEXT)
  • lat (NUM)
  • lon (NUM)
  • isPublic (BOOL)
